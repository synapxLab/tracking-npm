🎯 PROCHAINES ÉTAPES
════════════════════

1️⃣  Extraire l'archive
    └─> tar -xzf tracking-npm-ready.tar.gz
    └─> cd tracking-npm

2️⃣  Initialiser Git
    └─> git init
    └─> git add .
    └─> git commit -m "feat: initial release v1.0.0"
    └─> git remote add origin git@github.com:synapxLab/tracking-npm.git
    └─> git push -u origin main

3️⃣  Installer et builder
    └─> npm install
    └─> npm run build

4️⃣  Publier sur NPM
    └─> npm login
    └─> ./publish.sh

5️⃣  Vérifier
    └─> https://www.npmjs.com/package/@synapxlab/tracking-npm
    └─> npm install @synapxlab/tracking-npm
